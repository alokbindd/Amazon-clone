# Amazon Clone

This project is a front-end clone of the Amazon homepage, built using HTML and CSS. It aims to replicate the look and feel of Amazon's main landing page, including navigation, product sections, and responsive design elements.

## Features

- **Navigation Bar:** Includes logo, address selection, search bar, language selection, sign-in, returns, and cart.
- **Hero Section:** Promotional banner with a link to Amazon India.
- **Shop Sections:** Multiple product boxes showcasing categories like gaming, home essentials, fashion, kitchen, and more, each with images and links.
- **Top Picks & Best Sellers:** Image carousels for top picks and best sellers in various categories.
- **Sign-in Prompt:** Section for personalized recommendations and new customer sign-up.
- **Footer:** Contains links to company info, making money with Amazon, payment products, help, and legal information. Includes a custom SVG for privacy choices.

## Project Structure

```
Amazon_clone/
│
├── index.html         # Main HTML file
├── style.css          # Main CSS file
└── images/            # All image assets
    ├── box1/          # Images for home essentials
    ├── box2/          # Images for fashion
    ├── box3/          # Images for new home arrivals
    ├── box4/          # Images for refresh your space
    ├── box5/          # Images for kitchen appliances
    ├── BS/            # Best Sellers images
    ├── single/        # Single-image sections (e.g., gaming, holiday gifts)
    └── topicks/       # Top picks images
```

## How to Run

1. Clone or download the repository.
2. Open `index.html` in your web browser.
3. All styling is handled by `style.css`. Make sure the `images` folder is in the same directory as `index.html`.

## Customization

- You can replace images in the `images/` subfolders to update product categories or banners.
- Modify `style.css` to change the look and feel or make the site more responsive.

## Dependencies
